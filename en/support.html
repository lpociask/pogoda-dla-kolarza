Common topics

No route forecast

Check your network connection and switch the weather source to Demo to confirm that the route file itself is valid.

Apple Weather / WeatherKit

Apple Weather requires active WeatherKit configuration for the app identifier. If Apple Weather does not return a forecast, Auto mode can try the next provider.

File import

Spin or Stay supports GPX and FIT files. A route file must contain at least two route points.
